Lung cancer is among the top reasons for
cancer deaths. Early detection of lung cancer improves the
chances of survival. With the advent of deep learning,
automated medical image classification systems have gained
popularity in medical diagnosis. This paper proposes a
Convolutional Neural Network (CNN) model using VGG16
transfer learning for multi-class classification of lung cancer
images. The proposed model has been trained on an 80:20
split train and test dataset using various data augmentation
methods. The experimental results show that the proposed
method has been able to classify the lung cancer images with
an overall accuracy of 80.8%. The results indicate that deep
learning models using transfer learning can be used for lung
cancer diagnosis in medical images.
